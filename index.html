About Me

I am an aspiring data engineer who loves organizing data and building apps to increase efficiency. I enjoy using technology to improve and automate workflows.

I first learned how to program as an accountant with the Texas Comptroller's Office. I began by experimenting with improving some of the more tedious processes in spreadsheets. I discovered that I could use Visual Basic for Applications to automate some of my repetitive tasks. From that point on, I was hooked on coding. Since then, I have taught myself Java, Python, Pandas, Numpy, SQL, HTML, CSS, PHP, and more. I have also learned how to increase efficiency in my previous real estate business using automation tools such as Citrix Workflow Automation, Zapier, Process St, etc.

I am thankful we live in an age where so much education and opportunity are available for those willing to work for it. I have always considered myself to be a life-long learner. So, I can't wait to learn new things. As a matter of fact, I am not waiting. I am currently teaching myself about advanced SQL operations and how to use JWT to authenticate users in Flask.

My Projects

ETL-MRTS-Data-Analysis

The ETL-MRTS-Data-Analysis project shows how to create an ETL pipeline of the "Monthly Retail Trade Survey" dataset from the U.S. Census Bureau. The project uses Python, Pandas, SQLAlchemy, and MYSQL to construct the pipeline. Then the data is analyzed using Matplotlib.

VerseKeeper App

VerseKeeper is a user-friendly Bible memorization app under development for the Android platform. VerseKeeper allows users to import, manage, practice, review, schedule, and research their favorite Bible verses.

The app was developed using Java and SQLite. It also utilizes a variety of libraries and services, such as:

  • jsoup: for web scraping
  • Firebase API: for user authentication
  • OkHttp: an HTTP client
  • Retrofit: a Rest client
  • lib-recur: a recurrence processor

VerseKeeper is on a private Github repository, so I have included some screenshots of the following topics:

  • web scraping
  • user authentication
  • API access

Motivated Seller Mailing List

The Motivated Seller Mailing List project is a significant update to a workflow used in my previous real estate business to market to motivated sellers. The previous version primarily used Microsoft Access and Excel. This version now uses Python, Pandas, MYSQL, and Flask.

The project extracts, transforms, and loads delinquent property tax records and code compliance data. Then merges it with external data to create lists of people who are most likely to sell their homes.

Docker-DB-CRUD

Docker-DB-CRUD is a small Python-based project that shows how Docker containers work well with popular SQL and NoSQL databases. The program uses recursive timers to insert timestamps in MYSQL. Then the program reads and inserts the data into MongoDB, Redis, and Cassandra sequentially.
