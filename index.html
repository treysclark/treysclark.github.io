About Me

I am a data engineer who loves organizing data and building apps to increase efficiency. I also enjoy using technology to improve and automate workflows.

I first came across programming when I was a sales tax auditor with the Texas Comptroller's Office. I discovered that I could use Visual Basic to automate some of the repetitive tasks I used to do by hand in Excel. From that point on, I was hooked on coding. Since then, I have become experienced in SQL, Java, Python, Pandas, Numpy, Matplotlib, HTML, CSS, and more.

I have provided a brief overview of some of my recent projects below, and I have more detailed explanations of my projects on Github.

My Projects

MRTS Dataset: ETL & Data Analysis

The MRTS Dataset: ETL & Data Analysis project creates an ETL pipeline of the "Monthly Retail Trade Survey" dataset from the U.S. Census Bureau. The project uses Python, Pandas, SQLAlchemy, and MYSQL to construct the pipeline. Then the data is analyzed using Matplotlib.

Predicting Housing Prices

The Predicting Housing Prices project uses machine learning to show how housing prices can be predicted with at least 90% accuracy using publicly available datasets and software. It uses Pandas and Numpy to extract and transform the data, Matplotlib to explore the data, and scikit-learn to provide predictive analysis.

VerseKeeper App

VerseKeeper is a user-friendly Bible memorization app under development for the Android platform. VerseKeeper allows users to import, manage, practice, review, schedule, and research their favorite Bible verses.

I created a website to showcase all of VerseKeeper's features.

The app was developed using Java and SQLite. It also utilizes a variety of libraries and services, such as:

  • jsoup: for web scraping
  • Firebase API: for user authentication
  • OkHttp: an HTTP client
  • Retrofit2: a Rest client
  • lib-recur: a recurrence processor

VerseKeeper is on a private Github repository, so I have included some screenshots of the following topics:

  • web scraping
  • user authentication
  • API access

Docker-DB-CRUD

Docker-DB-CRUD is a small Python-based project that shows how Docker containers work well with popular SQL and NoSQL databases. The program uses recursive timers to insert timestamps in MYSQL. Then the program reads and inserts the data into MongoDB, Redis, and Cassandra sequentially.

Motivated Seller Mailing List

The Motivated Seller Mailing List project is a significant update to a workflow used in my previous real estate business to market to motivated sellers. The previous version primarily used Microsoft Access and Excel. This version now uses Python, Pandas, MYSQL, and Flask.

The project extracts, transforms, and loads delinquent property tax records and code compliance data. It then merges it with external data to create lists of people who are most likely to sell their homes.

TreySClark Webpage

This website started with a simple readme file on Github, but it didn't stop there! It now uses a static HTML5 template that is responsive and uses the Bootstrap library.

In addition to making significant changes to the template, I have also included other responsive design sections, such as the skills section, which uses a clickable hexagon grid. I even used Adobe Illustrator to extract SVG code so I could more easily make changes to images. I also used a submittable form using Basin, which is a static form endpoint solution.

It was a lot of fun making this website. I can't wait to highlight the steps I have taken to date.
